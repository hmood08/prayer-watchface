تثبيت Android Studio

مرة واحدة فقط — لا تحتاج تكرارها

1
حمّل Android Studio من الموقع الرسمي — مجاناً تماماً
▼
🔗
اذهب لهذا الرابط وحمّل النسخة المناسبة لجهازك (Windows / Mac):
developer.android.com/studio
Android Studio Setup
📦 Downloading Android Studio Meerkat 2024.3.2...
الحجم: ~1.2 GB — الوقت: 5-15 دقيقة حسب سرعة الإنترنت
💡
أثناء التثبيت: اقبل كل الخيارات الافتراضية، وعند سؤالك عن نوع التثبيت اختر Standard
2
ثبّت Wear OS Emulator (محاكي) لتجربة الساعة بدون توصيلها فعلياً — اختياري
▼

بعد فتح Android Studio لأول مرة:

🔧 Tools → Device Manager
➕ Create Virtual Device
⌚ Wear OS → Galaxy Watch 6 (40mm)
▶ Next → Tiramisu API 33 → Finish

توصيل الساعة وتشغيل المشروع

فعّل وضع المطور على الساعة أولاً

5
فعّل Developer Mode على الساعة اضغط على رقم البناء 7 مرات متتالية
▼
⚙️ الإعدادات
↳ معلومات الساعة
  ↳ معلومات البرنامج
    ↳ رقم البناء — اضغط 7 مرات 👈
✓ تم تفعيل وضع المطور!
💡
بعد التفعيل ستظهر قائمة جديدة "خيارات المطور" في الإعدادات — ادخل إليها وفعّل ADB Debugging
6
وصّل الساعة بالكمبيوتر عبر WiFi (نفس الشبكة) أو USB
▼

طريقة WiFi (الأسهل):

📶 الإعدادات → الاتصالات → WiFi
✓ تأكد أن الساعة والكمبيوتر على نفس الشبكة
🔢 خيارات المطور → ADB over WiFi → انسخ IP:PORT
192.168.1.XXX:5555

في Android Studio:

🔧 Tools → Device Manager → Pair using Wi-Fi
📝 أدخل IP:PORT من الساعة
✓ Connect — ستظهر الساعة في القائمة
7
اضغط ▶ Run وجه الساعة سيظهر على الساعة فوراً!
▼
📱 الجهاز المتصل: Galaxy Watch 6 (ADB)
▶ Run 'app'  (أو Shift+F10)
✓ Installing on device... Watch face set as active!
🎉
بعد نجاح التشغيل، ارفع معصمك — ستجد وجه الساعة الجديد يعمل مباشرة!
٤

ربط أوقات الصلاة (Complications)

8

▼
⌚ اضغط مطولاً على وجه الساعة في الساعة
✏️ Customize / تخصيص
🕌 Complication 1 ← اختر Muslim Pro → "الصلاة الحالية"
🕌 Complication 2 ← اختر Muslim Pro → "الصلاة التالية"
⚠️
إذا لم يكن تطبيق أوقات الصلاة مثبتاً على هاتفك، ستظهر بيانات البطارية كبديل تلقائي حتى تثبّته.
